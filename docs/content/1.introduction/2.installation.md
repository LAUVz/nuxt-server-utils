
# Installation

1. Install the module:

::code-group

  ```bash [npm]
  npm install nuxt-server-utils
  ```

  ```bash [yarn]
  yarn add nuxt-server-utils
  ```

  ```bash [pnpm]
  pnpm add nuxt-server-utils
  ```
::

2. Add `nuxt-server-utils` to the `modules` section of `nuxt.config.ts`:

::code-group

  ```ts [nuxt.config.ts]
  export default {
    modules: [
      'nuxt-server-utils',
    ],
  }
  ```
::

You're ready to go!
